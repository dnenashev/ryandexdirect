# Changelog

## 2026-02-26 — Еженедельный экспорт статистики Яндекс.Директ → Google Sheets

### Что добавлено

Автоматический сбор статистики по рекламным кампаниям Яндекс.Директ и экспорт в Google Sheets. Запускается каждый понедельник по cron (GitHub Actions) или вручную.

### Новые файлы

| Файл | Описание |
|------|----------|
| `scripts/weekly_campaign_stats.R` | R-скрипт сбора и экспорта статистики |
| `.github/workflows/weekly-stats.yml` | GitHub Actions workflow (cron каждый пн 07:00 UTC) |
| `docs/DEPLOY.md` | Инструкция по деплою |
| `docs/project-overview.md` | Архитектура проекта с Mermaid-диаграммами |

### Изменённые файлы

| Файл | Что изменено |
|------|-------------|
| `DESCRIPTION` | Добавлен `httr2` в Imports (исправлен баг `R CMD check`) |

### Возможности скрипта

- **Мульти-аккаунт**: поддержка нескольких аккаунтов Яндекс.Директ через секрет `YANDEX_ACCOUNTS` (JSON-массив)
- **Паспортные организации**: работает с аккаунтами вида `porg-xxx` (требуется токен с правом `passport:business`)
- **Календарные недели**: статистика формируется строго за полную неделю пн–вс
- **Накопление данных**: каждый запуск дозаписывает новую неделю, повторный запуск за ту же неделю обновляет данные
- **Одна вкладка**: все данные в листе «Статистика»

### Метрики в таблице

| Столбец | Источник |
|---------|----------|
| Неделя | `W08 (16.02–22.02)` |
| Аккаунт | Логин организации |
| Кампания | Название кампании |
| Показы | Impressions |
| Клики | Clicks |
| Расход, ₽ | Cost (с НДС) |
| CTR, % | Клики / Показы × 100 |
| CPC, ₽ | Расход / Клики |
| Конверсии | Conversions |
| CPA, ₽ | Расход / Конверсии |
| Ср. объём трафика | AvgTrafficVolume |

### Секреты GitHub Actions

| Секрет | Описание |
|--------|----------|
| `YANDEX_ACCOUNTS` | JSON-массив: `[{"login":"porg-xxx","token":"y0__..."},...]` |
| `GOOGLE_SHEET_ID` | ID Google-таблицы из URL |
| `GOOGLE_SA_KEY` | JSON-ключ сервисного аккаунта Google |
